# Query Blocks
Put your search queries into `query` code blocks and have them displayed like if they were in the search bar.

Can use the same filters, like `tag:`, `path:`, `file:`, etc.

## Example
Seraching for files in path:
```query
path:Basics/
```

# Youtube videos

```query
	tag:#video
```
